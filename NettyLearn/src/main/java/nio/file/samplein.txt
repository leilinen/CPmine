“明明是三个人的电影，我却始终不能有姓名？”我相信刘若英对那段感情付出了很多，也做了很多挣扎。但是对陈升老婆而言，刘若英就是癞蛤蟆一样的存在，不咬人膈应人，她又何辜遭遇丈夫被全世界心疼婚外情？有人相信陈升可能根本不爱刘若英吗？又或者陈升离婚跟刘若英在一起，会有几个人觉得刘若英是小三儿？陈升的老婆是不是就该大方的让地方？这种世人皆知的踩在别人头上的爱而不得还有怀着白月光前任应付现任的最恶心不过了。记得她有首歌叫《成全》，我想说清醒一点吧，不是你成全了陈升和他老婆，陈升这个人本身就是他老婆的。或者说，陈升选择的是老婆孩子，在他心里这两个人比刘重要。如果电影里男主或者女主是忘不了前任还吊着现任，那我绝对不会去看的，如果只是互相放不下，没有任何羁绊的去互相寻找那我觉得可以一看。就算是对那段感情的一段幻想了。至于刘陈那些陈年旧事，两个已婚的人，能不能给陈升老婆还有刘若英丈夫一点净土？如果刘若英那这段事炒电影的话，那只能说，这些年看错了一个人，我以为的奶茶终究是个绿茶。
